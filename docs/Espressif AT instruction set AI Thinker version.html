1 Overview
Describe the function and usage of Espressif AT command set
The instruction set is mainly divided into: basic AT commands, Wifi function AT commands, TCP / IP toolbox AT commands
Order etc.
AT is compiled based on esp_iot_sdk, please copy all contents in the folder "at" to the corresponding
Compile in the "app" folder of the version esp_iot_sdk. As shown below
Download bin to flash:
boot.bin, flash to flash 0x00000 user1.bin, flash to flash 0x01000
blank.bin, burn to flash 0x3E000 and 0x7E000 to realize the initialization of parameters.

2 Instructions
Each instruction can be subdivided into four commands
Test command
AT + <x> =?
This command is used to query the parameters set by the setup command or internal program to
And its value range.
Query command
AT + <x>?
This command is used to return the current value of the parameter.
Set command
AT + <x> = <…> This command is used to set user-defined parameter value.
Excuting an order
AT + <x>
This command is used to execute variable parameters controlled by the internal program of the module.
Variable functions.
Note: 1. Not every AT command has the above 4 types of commands
2. The data in [] is the default value, it does not need to be filled in or may not be displayed
3. Use double quotes to represent string data. "String".
For example, AT + CWSAP = "ESP756290", "21030826", 1,4
4. Baud rate 115200
5. Enter "\ r \ n" ending with carriage return and line feed

AT + UART set serial port configuration
Set command
AT + UART = <baudrate>,
<databits>, <stopbits>,
<parity>, <flow control>
response
OK
Parameter Description
<baudrate> serial port baud rate
<databits> data bits
5: 5 bit data bits
6: 6 bit data bits
7: 7 bit data bit
8: 8 bit data bit
<stopbits> stop bits
1: 1 bit stop bit
2: 1.5 bit stop bit
3: 2 bit stop bit
<parity> check digit
0: None
1: Odd
2: EVEN
<flow control> Flow control
0: Disable flow control
1: Enable RTS
2: Enable CTS
3: Enable RTS and CTS at the same time
Explanation
1. This setting will be saved in Flash user parameter area, after power on again
Still in effect.
2. The use of flow control requires hardware support for flow control, MTCK is UART0
CTS, MTDO is UART0 RTS
3. Baud rate support range: 110 ~ 115200 * 40
Examples
AT + UART = 115200,8,1,0,3
